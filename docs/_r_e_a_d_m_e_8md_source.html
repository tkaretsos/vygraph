14 ## How to build
15 
16 vygraph is using the Clang's build system. Read more [here](http://clang.llvm.org/get_started.html).
17 I provide two scripts to make things a bit simpler, because vygraph's source directory lies under a
18 different directory from the LLVM/Clang source code. Here are the two scripts:
19 
20 #### download_llvm
21 This script is only for downloading and extracting the required LLVM and Clang sources. If you already
22 have the sources of LLVM, Clang and [Clang tools](http://clang.llvm.org/docs/ClangTools.html), it is not required to execute this script.
23 
24 `sh download_llvm <directory>`
25 
26 where `<directory>` is the directory where you want the sources to be downloaded and extracted.
27 When this script is executed, it will generate the file `llvm_dirs` which contains the paths for
28 the source, build and bin directories of LLVM. You can use this file with the next script, in order
29 to configure the project.
30 
31 #### configure
32 This script helps set the current project as a tool project of Clang, so when you build LLVM and Clang
33 this project will be built as well. When you run this script you need to provide it with three directories.
34 The LLVM source directory, the LLVM build directory and the LLVM bin directory:
35 
36 `sh configure <llvm_src_dir> <llvm_obj_dir> <llvm_bin_dir>`
37 
38 If you previously ran the `download_llvm` script you can also:
39 
40 `sh configure $(cat llvm_dirs)`
41 
42 After `configure` script is executed successfully you can run `make` in the project directory and LLVM,
43 Clang, Clang tools and this project will be built.
44 
45 #### Example
46 If you do not have the sources of LLVM and Clang:
47 
48 `sh download_llvm ..`
49 `sh configure ($cat llvm_dirs)`
50 `make`

53 ## How to run
54 
55 After the project has been successfully built, a symlink to the executable will be created in the project's
56 root directory (`vygraph`). Execute it and give a C source code file as an input:
57 
58 `./vygraph my_file.c --`
59 
60 The two dashes in the end are required by the tool.
