    void analyze ( const clang::FunctionDecl *  funcDecl)

    Analyzes the function that is about to be translated.

    Assigns the required location numbers to the C statements of the function, when this is necessary. At the same time, it checks for the correctness of the atomic regions.

    Parameters
    funcDeclThe reference to the function
